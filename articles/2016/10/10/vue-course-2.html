8.组件：
1.注册组件：
    1). 扩展：var fcc = Vue.extend({template:"<span>fcc</span>"})
    2).注册：Vue.component('my-fcc',fcc)
    3).实例：new Vue();
    4).使用：在html中插入<my-fcc></my-fcc>，会将注册的组件替换到当前位置，即渲染后：<span>fcc</span>

2.局部注册：
    在组件内注册其他组件称为局部注册，局部注册的组件只能在父组件中使用；
    var child = Vue.extend(...)//子组件扩展
    var parent = Vue.extend({
        template : "",
        components : {
            "my-child" : child //局部注册在父组件内
        }
    })

3.语法糖：
    注册组件1,2步可以合为：
    Vue.component('my-fcc',{template:"<span>fcc</span>"})

4.组件选项问题：
    el，data的值必须是函数，函数中返回才是他们的最终值。

5.模板解析问题：
    由于某些标签内部只允许存在特定的标签，因此在使用组件自定义的标签时，就会出现问题。这时候使用is特性。

6.组件之间使用props传递数据：
    在注册组件时，组件之间约定一个属性名：
    Vue.component({
        props : ['message','name']
    })
    然后再父组件中使用该属性进行传值。
    <child message="hello" name="fcc"></child>
